YumTask is a client-side web application that combines task management with food and meal planning organization. It helps users manage daily responsibilities while promoting structured and healthy lifestyle habits.
Features
•	Add, edit, delete, and complete tasks
•	Real-time analytics dashboard with Chart.js
•	Latest activity tracking
•	Dark mode with saved preference
•	Fully responsive design using Bootstrap 5
•	LocalStorage data persistence (no backend required)
•	Contact form with validation and confirmation modal
•	Homepage carousel for feature presentation
•	Technologies Used
•	HTML5
•	CSS3
•	Bootstrap 5
•	JavaScript (Vanilla ES6)
•	Chart.js
•	LocalStorage API
Concept
YumTask is designed as a lightweight productivity assistant focused on food-related planning. It allows users to organize cooking tasks, grocery lists, and meal preparation schedules while tracking completion progress visually.
