# YumTask

A web-based Task Management application that allows users to:

- Add, edit, and delete tasks
- Filter tasks by status and priority
- Sort tasks by name or due date
- Track task analytics with charts

## Features

- Responsive design using Bootstrap
- Dark mode toggle
- Accessible HTML with proper ARIA labels
- Task analytics with Chart.js
